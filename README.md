# 세무직 합격 플래너

국어, 영어, 회계학, 세법의 공부 기록과 오답 복습, 영어 단어 DAY, 학습 통계를 한곳에서 관리하는 반응형 웹앱입니다.

## Windows 앱 실행 방법

바탕화면의 `세무직 합격 플래너` 아이콘을 더블클릭하면 독립된 앱 창으로 실행됩니다.

- 주소창이나 브라우저 메뉴 없이 전용 앱 창으로 열립니다.
- 입력한 데이터는 앱 전용 `AppData` 폴더에 자동 저장됩니다.
- 앱을 업데이트해도 `AppData` 폴더를 지우지 않으면 기록이 유지됩니다.

## 모바일 앱 설치

이 폴더를 HTTPS 웹 주소에 올린 뒤 휴대폰에서 접속하면 설치할 수 있습니다.

- 안드로이드: 화면의 `설치` 버튼 또는 Chrome 메뉴의 `앱 설치`
- 아이폰: Safari의 공유 버튼 → `홈 화면에 추가`
- 설치 후에는 주소창 없는 앱 화면으로 실행됩니다.
- 한 번 실행한 뒤에는 인터넷이 없어도 사용할 수 있습니다.
- 기록은 설치한 휴대폰 안에 저장되며 다른 기기와 자동 동기화되지는 않습니다.

## 꾸미기 · 시험 설정

- 포레스트, 네이비, 퍼플, 로즈, 브라운 테마와 사용자 지정 색상을 지원합니다.
- 카메라 권한을 허용해 직접 촬영하거나 사진 앨범에서 배경 이미지를 선택할 수 있습니다.
- 선택한 사진은 해당 기기 안에만 저장되며 외부 서버로 전송되지 않습니다.
- 세무직, 일반행정직, 교육행정직, 검찰직, 교정직, 관세직 추천 과목을 제공합니다.
- 과목을 직접 추가·삭제하고 과목별 플래너 색상을 변경할 수 있습니다.

## 영어 단어와 스마트 시험

- 각 DAY 카드에서 완료 여부를 바로 체크할 수 있습니다.
- DAY 번호와 기록을 수정하거나 휴지통 버튼으로 삭제할 수 있습니다.
- 헷갈린 단어는 `영단어 = 뜻` 형식으로 저장합니다.
- 저장한 단어에서 영단어→뜻, 뜻→영단어, 혼합 시험을 자동 생성합니다.
- 시험 정답과 오답 횟수는 해당 DAY 기록에 누적됩니다.

## 오답 기반 스마트 문제

- 오답의 과목, 세부영역, 문제 출처, 틀린 이유와 메모를 분석해 회상형 문제를 만듭니다.
- 모범 답안을 확인한 뒤 `기억했어요` 또는 `다시 복습`으로 평가할 수 있습니다.
- 다시 복습을 선택한 오답은 다음 날 복습 일정으로 자동 등록됩니다.
- 분석과 문제 생성은 기기 안에서 이루어지며 오답 내용이 외부로 전송되지 않습니다.

## 오늘 해야 할 일

- 홈에서 오늘 할 일을 과목, 예정 시간과 함께 추가할 수 있습니다.
- 완료 체크, 내용 수정, 휴지통 이동을 지원합니다.
- 완료율은 홈 화면에서 바로 확인할 수 있습니다.

## 커리큘럼 자동 분배형 스터디 플래너

- `커리큘럼 관리`에서 과목, 세부영역, 커리큘럼 이름, 공부유형, 전체 분량, 완료한 분량, 단위, 목표/현재 회독 수, 시작일, 마감일, 우선순위, 예상 소요시간을 입력할 수 있습니다.
- 전체 분량과 목표 회독 수를 기준으로 남은 분량, 남은 날짜, 하루 평균 필요 분량을 자동 계산합니다.
- `오늘의 자동 계획`에서 최소 목표, 권장 목표, 초과 목표를 확인할 수 있습니다.
- 공부가 밀리면 남은 기간과 남은 분량을 다시 계산해 하루 목표량이 자동 조정됩니다.
- 오늘 완료하지 못한 스터디 플래너 항목은 다음 날 이월 항목으로 표시됩니다.
- 하루 공부 가능 시간보다 계획량이 많으면 과부하 경고를 보여줍니다.
- `스터디 플래너`에서 자동 계획을 시간표로 배치하고 완료 체크를 하면 커리큘럼 진행률에 반영됩니다.

## 기록 수정과 휴지통

- 공부 기록, 오답·복습, 영어 단어 DAY, 오늘 할 일, 커리큘럼, 스터디 플래너를 수정할 수 있습니다.
- 삭제한 기록은 통합 휴지통으로 이동합니다.
- 휴지통에서 기록을 복원하거나 완전히 삭제할 수 있습니다.

## PC · 모바일 데이터 연동

- `꾸미기 · 설정`의 `백업 파일 내보내기`로 모든 기록을 JSON 파일에 저장합니다.
- 다른 기기의 `백업 파일 불러오기`에서 파일을 선택하면 기록이 옮겨집니다.
- GitHub Pages는 서버가 없는 정적 앱이므로 실시간 자동 동기화가 아닌 안전한 백업 파일 방식입니다.
- 가져오기 전 기존 데이터는 브라우저 내부에 임시 백업됩니다.
- 홈의 집중 타이머에서 과목과 세부영역을 고른 뒤 공부를 시작할 수 있습니다.
- 일시정지한 시간은 제외되며, 종료하면 순공시간으로 자동 저장됩니다.
- 오늘의 기록은 과목별 색상으로 컬러 플래너에 표시됩니다.
- 처음 실행할 때는 화면 구성을 확인할 수 있도록 예시 데이터가 표시됩니다.
- 상단의 `공부 기록` 버튼과 각 화면의 추가 버튼으로 새 기록을 입력할 수 있습니다.
- 시험일은 홈 화면의 `시험일 설정`에서 변경할 수 있습니다.

## 주요 화면

- 홈: D-day, 집중 타이머, 컬러 플래너, 오늘 공부시간, 목표 달성률, 과목별 시간, 복습 일정, 주간 흐름
- 공부 기록: 과목별 필터, 공부·순공시간, 문제 수, 정답률, 회독 수
- 커리큘럼 관리: 과목별 분량, 마감일, 회독 목표, 우선순위 관리
- 오늘의 자동 계획: 최소/권장/초과 목표, 이월 항목, 과부하 경고
- 스터디 플래너: 자동 시간표 생성, 완료 체크, 계획 수정
- 오답·복습: 복습 예정/완료 관리
- 영어 단어: DAY별 완료, 복습 횟수, 헷갈린 단어 수
- 통계: 주간 공부시간, 과목 비중, 정답률, 회독·오답 현황, 커리큘럼 진행률, 계획 달성률
